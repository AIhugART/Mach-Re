## Mapping Confidence Rules

Adapted from BE↔QM mapping schema, recalibrated for three-system axiom-level triangulation.

### Triangulation-not-equivalence warning

Cross-system correspondences are structural analogies only. High triangulation score = strong convergence of structural evidence from independent compass systems. It does NOT mean the compass systems are equivalent to each other, to the axiom, or to the object.

- `Strong` — Close structural/functional parallel with ≥2 independent node/edge codes.
- `Medium` — Clear structural analogy with partial scope match or single-node support.
- `Weak` — Nearest available comparison; compass system addresses the domain but lacks explicit structural parallel.
- `Gap` — Compass system has no concept addressing this axiom element.

### Triangulation score (X.X/3) interpretation

- **3.0/3** — All three Strong → strongest convergence
- **2.5/3** — Two Strong + one Medium → robust convergence
- **2.0/3** — Two Medium + one Weak, or one Strong + one Medium + one Gap → adequate
- **1.5/3** — Below threshold → requires explicit justification
- **<1.5/3** — Insufficient triangulation → axiom element is `[proposed-by-this-project]` with limited compass support

### Evidence/provenance discipline

- A-evidence: N_PN_XXXXX node codes + source page/line
- B-evidence: N_AW_ASH_XXXXX or N_AW_WEI_XXXXX + source line
- C-evidence: N_BE_XXXXX + source line
- Provenance tags: `[established]` = independently verifiable; `[proposed-by-this-project]` = MACH RE coinage; `[interpretation]` = mapping judgment

# PART IV — BOUNDARY STATEMENT

## What This Mapping Does NOT Claim / Những gì Mapping này KHÔNG tuyên bố

1. **This mapping does not claim identity** between Phan Ngọc's cultural theory, Ashby/Weick's cybernetics, Buddhist Epistemology, and MACH RE axioms. The compass systems are analytical lenses (thấu kính); they are not the object being described.
2. **This mapping does not claim that A, B, or C endorse MACH RE.** The correspondence is structural analogy identified by this project; the original authors did not intend their work to support MACH RE.
3. **This mapping does not introduce new canonical axioms.** All axiom statements are sourced from `axiom_spec.md` (single source of truth). This file only expands the evidence base for existing axioms.
4. **This mapping does not claim that any single compass system is sufficient.** MACH RE requires all three lenses — removing any one would collapse the triangulation, per `axiom_spec.md §0.3` (bất đối xứng phụ thuộc).
5. **This mapping does not claim empirical proof.** Triangulation is a structural convergence check, not experimental confirmation. The axioms remain falsifiable per their stated conditions in `axiom_spec.md`.
6. **This mapping respects the taxonomy of three roles** (lens/mirror/beam — `axiom_spec.md §0.4`): A·B·C = lens; Ubuntu = mirror; Tam giáo = reference beam. No role substitution is implied.
7. **Provenance is explicit:** Elements tagged `[proposed-by-this-project]` (e.g., III-orthogonality V ⊥ H, VI-C1 conditions, F-AND-gate) are MACH RE coinage — not established in any compass system and not yet peer-reviewed.
